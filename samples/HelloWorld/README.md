# HelloWorld (AiVectra)

Canonical baseline sample for AiVectra app structure and runtime usage.

## Behavior
Displays `Hello, World!` using AiVectra public API and standardized runtime flow.

## Run
```bash
./scripts/aivectra run ./samples/HelloWorld/
```

## Notes
- Sample code does not call `sys.*` directly.
- Platform metadata sources are TOML under `Targets/`.
